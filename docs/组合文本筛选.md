# 组合文本筛选的问题

## 问题 

假设有下面一段话，

``` text
1.单价：拾壹万肆仟壹佰玖拾伍元整（114,195元）; 总价：壹佰壹拾肆万壹仟玖佰伍拾元整（1,141,950元）
```

现在根据抽取出的实体去把位置信息选出来

``` text
拾壹万肆仟壹佰玖拾伍元整（114,195元）

壹佰壹拾肆万壹仟玖佰伍拾元整（1,141,950元）
```

## 实现思路

### 已知条件

1. 简化tree为数组

xml组成

```js
['1', '.', '单价：', '拾壹万肆仟壹佰玖拾伍', '整（1', '14,195', '元）', '; ', '总价：', '壹佰壹拾肆万壹仟玖佰伍拾', '元整（', '1,141,950元）']
```

2. 已知条件: 抽取实体的字符串

```text
拾壹万肆仟壹佰玖拾伍元整（114,195元）

壹佰壹拾肆万壹仟玖佰伍拾元整（1,141,950元）
```

3. 抽取的字符串在数组中必定是连续的

### 结果

组成

```js
['1', '.', '单价：', ['拾壹万肆仟壹佰玖拾伍', '整（1', '14,195', '元）'], '; ', '总价：', ['壹佰壹拾肆万壹仟玖佰伍拾', '元整（', '1,141,950元）']]
```
